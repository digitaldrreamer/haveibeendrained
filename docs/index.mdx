## The Problem

In 2024-2025, Solana wallet drainers stole **$300+ million** from **324,000+ users**. Current security solutions are:
- ❌ **Centralized** - Single points of failure
- ❌ **Slow** - Delayed threat detection
- ❌ **Incomplete** - Missing many attack patterns
- ❌ **Not community-driven** - No way for users to contribute

## Key Features

- ✅ **On-Chain Registry** - Immutable, verifiable drainer reports
- ✅ **Real-Time Analysis** - Instant wallet security scanning
- ✅ **Multi-Pattern Detection** - SetAuthority, approvals, known drainers
- ✅ **Community-Powered** - Permissionless reporting with anti-spam
- ✅ **Solana Actions (Blinks)** - Check wallets directly from Twitter/Discord
- ✅ **Free API** - Public API for developers with rate limiting
- ✅ **Open Source** - Transparent, auditable code

## Architecture

The system consists of:

1. **Anchor Program** - On-chain drainer registry using PDAs
2. **API Server** - Transaction analysis via Helius RPC
3. **Frontend** - Modern UI with real-time wallet analysis
4. **Shared Package** - TypeScript types and utilities
