### Installation

#### From Chrome Web Store (Recommended)

#### From Source (Development)

If you want to install the extension from source (for development or before the Chrome Web Store release):

<Steps>
<Step title="Get the extension files">

Clone the repository or navigate to the extension directory:

```bash
git clone https://github.com/digitaldrreamer/haveibeendrained.git
cd haveibeendrained/packages/browsers/chromium
```

</Step>

<Step title="Open Chrome Extensions page">

1. Open Google Chrome (or Edge, Brave, Opera)
2. Navigate to `chrome://extensions/`
   - Or: Menu (‚ãÆ) ‚Üí Extensions ‚Üí Manage extensions

</Step>

<Step title="Enable Developer mode">

1. Find the "Developer mode" toggle in the top-right corner
2. Toggle it **ON** (it will turn blue/highlighted)

</Step>

<Step title="Load the extension">

1. Click the **"Load unpacked"** button
2. Navigate to and select the `chromium` folder:
   ```
   packages/browsers/chromium
   ```
3. Click **"Select Folder"**

</Step>

<Step title="Verify installation">

1. The extension should appear in your extensions list
2. The extension icon should appear in your browser toolbar
3. Click the icon to open the extension popup

</Step>
</Steps>

### Configuration

#### Custom API Endpoint

For local development or custom API deployments:

1. Click the settings icon (‚öôÔ∏è) in the extension popup
2. Enter your custom API endpoint (e.g., `http://localhost:3001`)

## Troubleshooting

### Extension Not Loading

**Problem:** Extension doesn't appear after loading

**Solutions:**
- Make sure you selected the `chromium` folder (not the parent `browsers` folder)
- Verify `manifest.json` exists in the selected folder
- Check that all required files are present (popup.html, popup.js, popup.css)
- Try refreshing the extensions page (`chrome://extensions/`)

### Can't Find "Load Unpacked" Button

**Problem:** "Load unpacked" button is missing

**Solutions:**
- Ensure Developer mode is enabled (toggle should be ON and highlighted)
- Try refreshing the extensions page
- Make sure you're using Chrome 88+ or a compatible Chromium browser

### API Errors

**Problem:** Getting errors when checking addresses

**Solutions:**
- Verify your internet connection
- Check that the API endpoint is correct in settings
- Try the default endpoint: `https://api.haveibeendrained.org`
- Check the browser console for detailed error messages

### Extension Icon Not Appearing

**Problem:** Extension is installed but icon doesn't show in toolbar

**Solutions:**
- Click the puzzle piece icon (üß©) in Chrome toolbar
- Find "Have I Been Drained?" in the list
- Click the pin icon to pin it to the toolbar
- Or drag it to your preferred position

## Keyboard Shortcuts

- **Escape** - Close the popup
- **Ctrl/Cmd + V** - Paste address from clipboard
