## What are Solana Actions?

Solana Actions (also called "Blinks") allow you to interact with Solana applications directly from social media platforms like Twitter and Discord, without leaving the platform.

## Using Have I Been Drained Actions

### From Twitter/X

1. **Find a wallet address** in a tweet or post
2. **Click the "Check with Have I Been Drained" action**
3. **View results** without leaving Twitter
4. **Share results** if desired

### From Discord

1. **Share or find a wallet address** in a Discord message
2. **Click the action button**
3. **View security analysis** in Discord
4. **Take action** based on results

## Features

- **No app switching** - Check wallets without leaving the platform
- **Instant results** - Quick security checks on the go
- **Easy sharing** - Share results with others
- **Viral potential** - Help spread security awareness

## How It Works

<Steps>
<Step title="Wallet address detected">

When a Solana wallet address is shared, our action becomes available.

</Step>

<Step title="Click to check">

Click the "Check with Have I Been Drained" button that appears.

</Step>

<Step title="Analysis runs">

Our API analyzes the wallet in real-time (10-15 seconds).

</Step>

<Step title="Results displayed">

View risk score and recommendations directly in the platform.

</Step>
</Steps>
