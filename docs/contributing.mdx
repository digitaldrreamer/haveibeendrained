## Quick Start

<Info>
  **Prerequisites**:
  - Node.js version 19 or higher (or Bun v1.2+)
  - Docker and Docker Compose
  - Solana CLI (for Anchor development)
  - Anchor v0.32.1+
</Info>

For detailed setup instructions, see the [Local Setup](/contributing/local-setup) guide.

## Development Workflow

1. **Start infrastructure**: `docker compose up -d`
2. **Run API in watch mode**: `cd packages/api && bun run dev`
3. **Run frontend in watch mode**: `cd packages/frontend && bun run dev`
4. **Build Anchor program**: `cd packages/anchor && anchor build`
