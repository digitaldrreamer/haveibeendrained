# Have I Been Drained? Documentation

Documentation for Have I Been Drained? - The first decentralized, community-powered wallet security checker on Solana.

## Development

Install the [Mintlify CLI](https://www.npmjs.com/package/mint) to preview your documentation changes locally:

```bash
npm i -g mint
```

Run the following command at the root of your documentation directory:

```bash
mint dev
```

View your local preview at `http://localhost:3000`.

## Publishing

Install the Mintlify GitHub app from your [dashboard](https://dashboard.mintlify.com/settings/organization/github-app) to automatically deploy changes from your repo. Changes are deployed to production automatically after pushing to the default branch.
